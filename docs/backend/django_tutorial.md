API -> Application Programming Interface :
    In Django, we create APIs to allow frontend applications like React to get data from the backend using specific API endpoints.

frontend rendering kinds:
- Client-Side Rendering (CSR)
    Rendering happens in the browser
    JavaScript builds the page after loading
    Common in pure React apps
-Server-Side Rendering (SSR)
    The server generates the HTML first
    Browser receives a ready page
    Common in Next.js
